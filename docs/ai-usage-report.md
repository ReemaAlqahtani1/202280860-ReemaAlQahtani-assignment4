## 1. Tools Used & Use Cases

### ChatGPT

ChatGPT was used as a support tool during the development of the final portfolio website for:

- Suggesting professional portfolio improvements and final production-ready features
- Assisting in debugging JavaScript logic for filtering, sorting, and rendering systems
- Helping implement Formspree contact form integration without backend development
- Providing UI/UX improvement suggestions for social links, Education section, CV button, and Back to Top button
- Improving responsive design and mobile compatibility
- Supporting README writing, technical documentation, and final project organization
- Reviewing accessibility improvements and professional portfolio structure

It was mainly used for explanation, guidance, debugging support, and documentation improvement rather than direct code generation.

---

### GitHub Copilot

GitHub Copilot was used inside VS Code to:

- Suggest syntax completions
- Speed up repetitive JavaScript and CSS patterns
- Provide quick structural suggestions for rendering functions and UI components
- Support repetitive styling and layout improvements
- Help with JavaScript event handling and DOM rendering logic

All suggestions were reviewed, tested, and modified where necessary before implementation.
