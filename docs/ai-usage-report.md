# AI Usage Report

**Assignment 3 -- Advanced Portfolio Features & API Integration**

------------------------------------------------------------------------

## 1. Tools Used & Use Cases

### ChatGPT

ChatGPT was used as a support tool during development for:
-   Suggest features.
-   Assisting in debugging filtering, sorting, and project rendering
    logic.
-   Providing UI/UX improvement suggestions for dropdowns, buttons, and
    layout consistency.
-   Helping refine documentation formatting and project structure.
-   Recommending using GitHub API as it makes much sense with the portfolio.

It was mainly used for explanation, guidance, and troubleshooting rather
than direct code production.

------------------------------------------------------------------------

### GitHub Copilot

GitHub Copilot was used inside VS Code to:

-   Suggest syntax completions.
-   Speed up repetitive JavaScript and CSS patterns.
-   Provide quick structural suggestions for rendering functions and UI
    components.

All suggestions were reviewed, tested, and modified where necessary.

------------------------------------------------------------------------

## 2. AI-Supported Enhancements

AI assistance contributed to refining and improving certain features,
including:

-   Dynamic project filtering and sorting system
-   Grid/List view toggle with LocalStorage persistence
-   GitHub API integration using Fetch API
-   Enhanced custom dropdown styling
-   Improved responsive layout for project controls
-   Better accessibility and keyboard interaction support

These features were implemented after understanding the underlying
concepts and adapting them to fit assignment requirements.
